🧠 How It Works

DocuMind AI uses a multi-stage AI pipeline:
Documents: Upload → Text Extraction (pdfplumber / python-docx) → Google Gemini 2.0 Flash → Detailed Summary (20-30 lines) + NER + Sentiment
Audio Calls: Upload → Base64 Encoding → Gemini Multimodal STT → NLP Analysis → SOP Validation → Compliance Scoring

⚙️ Tech Stack
Backend:
Python, FastAPI, Uvicorn
AI Engine:
Google Gemini 2.0 Flash
Frontend:
Vanilla HTML/CSS/JS
Deployment:
Vercel, Railway, Docker

DocuMind AI — Full Documentation

📄 Document Analysis

  • Multi-format: PDF, DOCX, PNG, JPG support
  • Detailed AI Summary: 20-30 line comprehensive breakdown via Gemini 2.0 Flash
  • NER: Extracts people, organizations, dates, amounts
  • Sentiment: Positive / Negative / Neutral classification

📞 Call Center Compliance

  • Audio formats: MP3, WAV, M4A, FLAC, OGG
  • Languages: Tamil (Tanglish), Hindi (Hinglish), English
  • SOP Validation: Greeting → Identification → Problem → Solution → Closing
  • Analytics: Payment preference, rejection reason, sentiment
  • Compliance Score: 0.0–1.0 with FOLLOWED / NOT_FOLLOWED status

🔐 Authentication

Login with email/password or continue as Guest. API calls require x-api-key header.
